# LunaOS 🌙
LunaOS is an Android ROM based on LineageOS, focused on bringing new features while maintaining stability. (At least that's what we think, I hope we get what we promised...)

## Devices
- Motorola G52 (rhode)
- Motorola G42 (hawao)
- Motorola G32 (devon)
